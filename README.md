# divide-into-fewer-rectangles

少ない数の長方形に分ける

## 実行例

Input:  

```shell
py main.py
```

Output:  

```plaintext
横幅、縦幅のあとに図形を続けてください。
例：
7 5
..xxx..
xxxxx..
xxxxxxx
..xxxxx
...xxxx
7 5
..xxx..
xxxxx..
xxxxxxx
..xxxxx
...xxxx

RESULT
------
0011100
2211100
2211133
0011133
0005533
```
